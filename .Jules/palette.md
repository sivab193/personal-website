## 2026-04-09 - Adding external link visual cues and making lists clickable
**Learning:** When making cards or list items clickable implicitly, adding an explicit visual indicator like an external link icon on hover, or an underline on the title, reduces cognitive load and improves accessibility/navigability.
**Action:** Add `group/title`, `group-hover/title:underline`, and a fading-in external link icon when converting static text elements to clickable links within complex components.
